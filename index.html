Conceitos essenciais

Repositório (repo)
Local onde o histórico de versões do projeto é armazenado. Pode ser local (máquina) ou remoto (GitHub).
Commit
Uma 'foto' do estado do projeto. Contém autor, mensagem e um hash único.
Branch
Linha de desenvolvimento isolada. Ex.: main, feature/login.
Merge
Operação que junta o histórico de duas branches.
Pull request (PR)
Mecanismo do GitHub para revisar/mesclar mudanças entre branches remotas.
Clone / Fork
Clone: copia do repo remoto para seu ambiente. Fork: cópia do repo para sua conta no GitHub.
Remote
Referência para repositório remoto (ex.: origin).

Comandos rápidos

git clone <url>
                   git checkout -b feature/nova
                   git add .
                   git commit -m "mensagem"
                   git push origin feature/nova
                   # Para atualizar sua branch com main
                   git fetch origin
                   git merge origin/main
                   # Resolver conflitos, depois:
                   git add <arquivo>
                   git commit
                   git push
